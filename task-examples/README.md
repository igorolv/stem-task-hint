# task-examples

Сюда кладутся наборы олимпиадных задач для обкатки скилла (обычно PDF).

Сами файлы задач **не коммитятся** в репозиторий (личные / авторские
материалы) — см. `.gitignore`. Держи их локально в этом каталоге.

Текущий набор: «Летняя серия 2026» (50 задач: геометрия, теория чисел,
комбинаторика, алгебра). Если продолжаешь работу, а файла здесь нет —
запроси его у пользователя.
