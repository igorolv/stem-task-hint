# stem-task-hint

Скилл для LLM, который даёт дозированные подсказки к олимпиадным задачам по
STEM (математика, физика, химия, программирование). Цель — не решить задачу
за ученика, а дать минимальный «толчок», когда решение застряло.

Подсказка может быть: наводящим вопросом по условию, названием области/метода,
упрощённой задачей-аналогом или фрагментом теории — выстроенными в «лестницу»
по возрастанию спойлерности, чтобы ученик остановился, как только появилась
своя идея.

## Структура

```
skills/stem-hint/
  SKILL.md            # ядро: методология подсказок (лестница уровней,
                      # протокол «реши сам → проверь → подсказывай», честность)
  domains/math.md     # доменная справка по математике
  profile-template.md # опциональный профиль ученика (терминология, темы)
feedback/             # протоколы обкатки, результаты итераций, карточки
tools/                # шаблон интерактивной карточки для обкатки
task-examples/        # наборы задач (PDF; не коммитятся — личные материалы)
AGENTS.md             # как продолжить разработку в новой сессии
```

## Использование

Скилл не привязан к конкретному harness или LLM — это чистый markdown.

- **Claude Code / совместимые harness:** положить `skills/stem-hint/` в каталог
  скиллов; формат SKILL.md (frontmatter + markdown) — открытая конвенция.
- **Любая другая LLM:** вставить содержимое `SKILL.md` плюс подходящий файл из
  `domains/` системным промптом, затем дать задачу и попросить подсказку
  (диалоговый режим) или карточку подсказок (card mode).

## Обкатка

Скилл проверяется на реальных задачах. Ключевая идея протокола: карточки
оценивает ученик по **уже решённым** им задачам — так можно судить о верности
направления и спойлерности, не портя себе нерешённые задачи. Оценка идёт по
трём осям (направление / спойлерность 0–3 / «помогла бы?») через интерактивную
страницу с поуровневым раскрытием.

- Методология и форма оценки: [feedback/letnyaya-seriya-2026/protocol.md](feedback/letnyaya-seriya-2026/protocol.md)
- Результаты и история правок: [feedback/letnyaya-seriya-2026/results.md](feedback/letnyaya-seriya-2026/results.md)

**Статус:** пройдено 3 итерации; обкатаны задачи №3 (геометрия) и №50 (алгебра).
Из фидбека выросли: протокол верификации, терминологические расшифровки,
правило честности (пример не доказывает — нужна атрибуция), лестница переменной
длины. Дальше — задачи из теории чисел и комбинаторики.

## Продолжение разработки

Если ты ИИ-агент, продолжающий проект, — начни с [AGENTS.md](AGENTS.md): там
быстрый старт, процесс сборки карточки и текущее состояние.

## Планы

- Доменные справки для физики, химии, программирования.
- Возможно, скилл-«роутер» для выбора доменного модуля — если доменные модули
  разрастутся. Пока классификация — один шаг внутри ядра, задачи «на стыке»
  просто подгружают два модуля.
