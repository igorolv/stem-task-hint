Задача 50 — карточка подсказок

Ты уже решила эту задачу — поэтому именно ты можешь оценить подсказки. Представь, что решение не шло: открывай уровни по одному и после каждого отвечай на три вопроса. Комментарий необязателен, но полезнее всего. Подсказки ведут по одному из возможных путей — он не единственный: если ты идёшь другой дорогой, это не значит, что ошиблась.

Ненулевые числа a и b таковы, что уравнение
a(x − a)² + b(x − b)² = 0
имеет единственное решение. Докажите, что |a| = |b|.

Уровень 4фрагмент теории

Квадратное уравнение Ax² + Bx + C = 0 при A ≠ 0 имеет ровно одно решение в точности тогда, когда B² − 4AC = 0. Случай A = 0 надо разобрать отдельно: там уравнение линейное и имеет ровно один корень при B ≠ 0.
