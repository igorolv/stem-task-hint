# Подсказки — задача 33, пункт b) (Летняя серия 2026) · v2

Сначала на доске написано число 312. За ход можно прибавить к числу или
вычесть из него произведение его цифр. Может ли через несколько ходов на
доске быть снова написано число 312?

**Домен:** математика (числовой процесс) | **Уверенность:** проверено

> В карточке нет решения. Читай по одному уровню и останавливайся, как
> только появилась своя идея.

## Уровень 1 — метод

Чётности здесь мало — раздели чётные числа тоньше: проследи за остатками
от деления на 4 вдоль ходов.

## Уровень 2 — структура

Ловушка — остаток 2: попав туда, число оттуда уже не выйдет — с одним
мелким исключением; найди его и проверь, что здесь оно не мешает. Для
доказательства самой ловушки выясни, когда произведение цифр чётного
числа не делится на 4, — и в этом случае посмотри на две последние цифры
самого числа.

---

**Приватный разбор (в карточку не идёт, здесь — для истории).**

*Чтение условия.* Условие задачи 33 верифицировано в этой же сессии при
подготовке пункта а) (извлечение pdfplumber + зум-рендер страницы 2
совпали дословно, кодпоинты — чистая кириллица, числа 141 и 312 сверены
визуально). Пункт b): старт 312, вопрос о повторном появлении 312.
Опечаток нет.

*Решение (ответ: нет).* Ход: n → n ± P(n). Ключ — поглощающий класс по
модулю 4: **«остаток 2 при делении на 4» — ловушка**. Доказательство:
пусть n ⩾ 10 и n ≡ 2 (mod 4). Если в n есть цифра 0 — P = 0, число
застыло (остаток не меняется). Иначе: n чётно ⇒ последняя цифра d ∈
{2, 4, 6, 8}. При d ∈ {4, 8} произведение цифр делится на 4. При d ∈
{2, 6}: n mod 4 = (10a + d) mod 4 = (2a + 2) mod 4, где a — цифра
десятков; n ≡ 2 ⇒ a чётна, а цифры 0 нет ⇒ a ∈ {2, 4, 6, 8} — второй
чётный множитель, и P снова делится на 4. В обоих случаях 4 | P ⇒
n ± P ≡ n ≡ 2 (mod 4). Старт: 312 ≡ 0 (mod 4), P(312) = 6, первый ход —
318 или 306, оба ≡ 2 (mod 4) (306 к тому же застывает: P = 0). Значит
после первого же хода число навсегда в классе «остаток 2», а 312 в нём
не лежит — 312 не появится. Формальная лазейка леммы — однозначные n = 2,
6 (там P(n) = n ≡ 2 mod 4) — недостижима: при n ⩾ 10 всегда
n − P(n) ⩾ 10 (для 10a + d: 10a + d − ad ⩾ 10).

*Проверка (численно).* (1) Поглощение «≡ 2 mod 4» — перебор n до 2·10⁶:
0 нарушений; n ⩾ 10 с n ≡ 2 и P(n) ≡ 2 (mod 4) не существует (0 случаев).
(2) n − P(n) ⩾ 10 при n ⩾ 10 — 0 нарушений. (3) Страховочный BFS из 312
на 60 ходов (значения до 10⁷): 312 не повторяется; все 831 достижимых
состояния ≡ 2 (mod 4). (4) Прообразы 312 (m ± P(m) = 312, m до 10⁵):
248 (≡ 0), 264 (≡ 0), 393 (≡ 1) — ни один не лежит в классе «остаток 2»,
т.е. недостижим после первого хода. Уверенность: **проверено**.

*Крукс — один:* поглощающий класс по модулю 4 (уточнение чётности из
пункта а) с неожиданной спецификой: ловушка — не «делится на 4», а
«остаток ровно 2»). Иных путей не видно: граф чётных из 312 бесконечен
(конечный перебор, как в пункте а), невозможен), других модулей/спусков
нет — v₂-формулировка («ровно одна двойка в разложении») — тот же класс
другими словами.

*Лестница из 2 уровней (метод → структура) — первая карточка, собранная
ПО правилу «одна ступень — один шаг» (итерация 10).* Дешёвой ступени ниже
метода не существует, и это новая разновидность анализа израсходованности
(правило линзы, итерация 9): всё дешёвое **сожжено соседним пунктом а)** —
рамка «ищи свойство-ловушку» израсходована решением а), чётность
израсходована её неудачей на b) (стало чётным навсегда — не различает),
эксперимент-дерево сожжён (застрявший ходил по чётным без видимой стены).
Несожжённое и самое дешёвое — конкретная линза «остатки mod 4»: ур.1 —
ровно один шаг (модуль назван, ловушка и доказательство не выданы; «чётности
здесь мало» — закрытие сожжённой ветки одной фразой, ит.7). Ур.2 — ровно
один шаг: стержень доказательства (класс-ловушка назван + пинованный
подвопрос «когда P чётного числа не делится на 4» + указатель «две
последние цифры»); сборка вывода и проверка старта остаются ученику.
После ур.2 добавить можно только саму выкладку = решение — лестница
кончается. Пин формы ответа (ит.10) на обеих ступенях: «проследи за
остатками вдоль ходов» (последовательность остатков), «когда P не делится
на 4» (условие на цифры). Ожидаемая монотонность: ур.1 = 2 → ур.2 = 3 —
честный высокий пол (класс задачи 35; занижать нечем — не имитируем).
Ответ («нет»), остаток 312 и значения первых ходов в карточке не названы.

*Множественные пути / fallback — НЕ применяются.* Путь сущностно один;
fallback-секции нет, пометки «путь не единственный» нет — осознанный
no-op механизма, шестой подряд (задачи 35, 26, 48, 33а, 47, 33b).
